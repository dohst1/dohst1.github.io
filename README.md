# AmazonSMP Website

This is the github repository for a website for me and my minecraft server, developed and maintained by Dohst (me).

This website contains information about the AmazonSMP Minecraft server, past seasons information, downloads to past seasons, information on Dohst's texturepacks/datapacks/mods, and some quotes.

If you would like to request bug fixs, changes or improvements please create a new issue on this github repo.

If you want to contribute yourself: i will only allow current AmazonSMP members to contribute; if you are a member and want to, then contact me on whatsapp/discord and i will give you permissions.
