# AmazonSMP Website

This is the github repository for a website for me and my minecraft server, developed and maintained by Dohst (me).

This website contains information about the amazonsmp Minecraft server, past seasons information, downloads to past seasons, information on Dohst's texturepacks/datapacks/mods, and some quotes.

If you would like to request bug fixes/changes/improvements please create a new issue on this github repository.

Contribution: i will only allow current amazonsmp members to contribute; contact me (dohst) on whatsapp/discord if you want to and i will give you permissions.
