# AmazonSMP Website

This is the github repository for a website for me and my minecraft server, developed and maintained by Dohst (me).

This website contains information about the AmazonSMP Minecraft server, past seasons information, downloads to past seasons, information and downloads for Dohst's texturepacks/datapacks/mods<!-- , and some quotes -->.

If you would like to request bugfixes, changes or improvements please create a new issue on this github repository, (if you are reporting an issue please specify which page(s) it applies to).
