About and Announcements

Features

Website Colors

You can change the color theme for the website from the 'Set Color' and 'Toggle Dark' buttons at the top of this page. Your settings will be saved for other website pages and the next time you visit this site.

Page Sections

All pages have toggleable sections, you can show and hide content by clicking on the headers (some tables can also be toggled by clicking on the header (first two rows)).

Updates

All updates are now on the Updates Page. They are sectioned for 'AmazonSMP', 'Website' and 'Dohst'. You should check the discord server for the latest updates.

Maintenance and Contribution

Report / Request

To report an issue or suggestion for the website, create a new issue on the github project.

Maintenance

The website will usually be updated every other week on friday at about 22:00 UTC. But it will at least be updated once every month.

The next planned updates are: 04/24, 05/08, 05/22, 06/19, 07/03.

Contribution

Only AmazonSMP Members will be allowed to contribute. If you would like to contribute to the website, please contact Dohst.

After your first contribution your name will be listed in the 'Website' section of the 'Credit' section on this page.
