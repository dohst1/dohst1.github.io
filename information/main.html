Download Instructions

For external full downloads (from mediafire), use small world download instructions.

Small Worlds (Seasons 3 to 8 (except 7))

  • Go to the Seasons table or the page of the season you want to download (left column of this page) and download the season you want
    (all the files are zipped and less than 100 MB).
  • When it finishes downloading, extract the zip file
    (the extracted folder should have a file inside called "level.dat").
  • Move that folder into your minecraft worlds folder
    (for windows it should be "%appdata%\.minecraft\saves").
  • When you open minecraft it should be in your singleplayer worlds.

Big Worlds (Seasons 9 to 12 (including 7))

  • Go to the Seasons table or the page of the season you want to download (left column of this page) and download all the season downloads you want
    (all the files are zipped and less than 100 MB).
  • When they finish downloading, extract all the zip files
    (the extracted folder ending in "_0" should have a file inside called "level.dat", i'll call it the main folder).
  • Inside the folder(s) not ending in "_0" there should be a folder called "region_x", move all the files from that folder to the main folder's "region" folder (repeat this for all the folders not ending in "_0").
  • Move the main folder into your minecraft worlds folder
    (for windows it should be "%appdata%\.minecraft\saves").
  • When you open minecraft it should be in your singleplayer worlds.

Other Worlds (Seasons 0, 1, 2, and Temporary 1, 2, 3, 4)

  • These worlds are unavailable as i have not saved the world files.

If you want any downloads that are not on this website, unfortunately i do not have any other saved world files (so i cant give you any others).
