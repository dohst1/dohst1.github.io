Rules:

  • No Cheating/Hacking (ban 4 - 16 days)
  • No Administrator Abuse (ban 3 - 8 days)
  • No Inappropriate Content (ban/mute 2 - 4 days)
  • No Griefing (ban 4 - 16 days)
  • No Spamming (mute 1 - 2 days)
  • No Assaulting Charlotte

Conditions:

  • No Administrator Abuse rule: Offender's admin will be removed, and wont be able to obtain admin2/3 again
  • No Inappropriate Content rule: Doesn't apply to insults unless repetitive. Offender will only be banned if victim wants to. (if banned and muted then will only add 1 count to total_bans_and_mutes)
  • No Griefing rule: Offender will only be banned if the victim wants to.
  • You can only vote if you played the server in the last 4 months (for over an hour).
  • Ban time = total_bans * current_offence_ban_time.
  • Mute time = total_bans_and_mutes * current_offence_mute_time / 2.
  • total_bans includes the current ban, and total_bans_and_mutes includes current mute and/or ban.
  • To add a new member send their minecraft username to an admin (Dohst, Mandoeam, Secret7, WhatCheeseBurger).
  • total_bans, total_bans_and_mutes, and total_mutes reset to 0 at the start of a new season (or part)
